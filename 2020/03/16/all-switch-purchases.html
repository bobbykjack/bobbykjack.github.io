Nintendo Switch Game Purchases

This is a list of all Nintendo Switch games that I’ve paid actual, cold, hard money for. So that excludes demos, games I’ve redeemed for things like IndieSelect, etc. Games bought with gold points are included; prices are after any discount for games on sale. The table can be sorted by clicking a column header.

There are 105 games here. This is far more than for any other Nintendo console I’ve owned (GB, SNES, Wii), mainly because I’ve never really bought digitally before, and therefore never really indie either. It’s addictive!

Best value for money? Has to be Hollow Knight. Worst? Probably Mario Kart 8 Deluxe.
